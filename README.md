# Actividad3_grupal_ascensor
Perfeccionamiento del sistema de control del ascensor industrial, mediante instrumentación avanzada
Este repositorio contiene el proyecto desarrollado en Wokwi para el perfeccionamiento del sistema de control del ascensor industrial ACME mediante instrumentación avanzada.
## Funcionalidades implementadas
- Control de cabina mediante servomotor.
- Selección de planta mediante pulsadores físicos.
- Medición de temperatura con sensor DHT22.
- Medición de iluminación mediante LDR.
- Detección de presencia mediante sensor PIR.
- Visualización local mediante pantalla LCD I2C.
- Control de iluminación mediante registro de desplazamiento 74HC595.
- Control remoto mediante receptor infrarrojo IR.
- Ajuste remoto de setpoint de temperatura.
- Ajuste remoto de setpoint de iluminación.
- Cambio entre modo automático y manual.
- Autodiagnóstico básico de sensores.
- Activación de modo seguro ante fallo.
## Componentes principales
- Arduino UNO
- Servo motor
- DHT22
- LDR
- PIR
- LCD I2C 16x2
- Receptor IR
- Mando IR
- Buzzer
- Registro 74HC595
- LEDs indicadores
